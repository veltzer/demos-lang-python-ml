# Exercise 05: Bar Chart

Given:

```
categories = ["apples", "oranges", "bananas", "pears", "grapes"]
counts     = [23, 17, 35, 12, 28]
```

1. Draw a vertical bar chart with `plt.bar`.
2. Set the title to `"Fruit counts"`.
3. Label the y-axis `"count"`.
4. Rotate the x-tick labels by 30 degrees so they don't overlap (use `plt.xticks`).
5. Save to `/tmp/05_bar_chart.png`.
