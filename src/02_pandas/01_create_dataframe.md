# Exercise 02: Create a DataFrame

Build a `DataFrame` from this dictionary:

```text
{
    "name":   ["Alice", "Bob", "Charlie", "Dana"],
    "age":    [30, 25, 35, 28],
    "salary": [70000, 50000, 90000, 65000],
}
```

Print:

1. the full DataFrame
2. its `shape`
3. its `columns`
4. the first 2 rows (using `head`)
