# Exercise 01: Basic Line Plot

Plot `y = x**2` for `x` in `0..9`.

1. Build the data with `numpy.arange` for `x` and a vectorized expression for `y`.
2. Use `plt.plot(x, y)` to draw the line.
3. Save the figure to `/tmp/01_line_plot.png` with `plt.savefig`.

Note: every solution in this folder uses `plt.savefig` instead of `plt.show()` so the
script can run without a display.
